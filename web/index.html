How To Read This Page

Current SOS
The recommended sum-over-states electronic polarizability data from this project, projected to Sr-87 hyperfine states.

A reference workflow that uses Sr1Pol/Sr-88 electronic alpha0, alpha1, alpha2 curves and applies the same Sr-87 Clebsch-Gordan projection.
Delta mF
The compared transition channel: 0 is a pi or same-mF channel, while +1 and -1 are common sigma channels. This is not the same as the trapping-light polarization geometry.
clean
A root that does not trigger this package's 3P2 near-resonance rule. It is not a guarantee of low experimental scattering.

Scope

  • The wavelength window is fixed to 495.5-534.5 nm. The page does not extrapolate outside this range.
  • Magic wavelengths are found from a 0.02 nm offline grid using adjacent-point linear interpolation.
  • Delta mF=0 is the same-mF special case; Delta mF=+1 and -1 are included for sigma-channel checks.
  • Elliptical or circular trapping light activates the vector polarizability contribution and can increase sensitivity to alpha1 data quality.
  • 3P2 roots must be reviewed with scattering, heating, and near-resonance distance before choosing an experimental operating point.

Source Policy

The main page intentionally displays one source at a time. Use the Source selector to switch between the project SOS result and the Sr1Pol-projected reference workflow. The two sources should be compared as alternative inputs, not averaged into one result.
